Just a sandbox where I test different mechanics for fun
Controls (mouse and keyboard / pad)
Movement:...............WSAD/Left Thumbstick
Aim:....................Mouse/Right Thumbstick
Light Attack:...........LMB/R1
Strong Attack:..........LMB/R1 (hold)
Throw:..................MMB/R2
Parry/Guard:............RMB/L1
Jump:...................Space/Button south
Pickup weapon:..........E/Button north
Interact:...............F/Button west
Change disguise:........F/Button west (next to a body)
Dodge:..................Shift/Button east
Noctovision:............N/R3
